# Jock.Net.TcpJson
A Simple Tcp &amp; Json Data Transport Library, base .Net Standard 2.0

## Install
**nuget** `Install-Package Jock.Net.TcpJson`

**cli** `dotnet add package Jock.Net.TcpJson`
